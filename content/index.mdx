## What is Canopy IIIF?

Canopy IIIF is an open-source static site generator for creating discovery-focused digital scholarship and collections websites with IIIF APIs. Canopy IIIF builds static sites that are fast, lightweight, and low-maintenance. Each generated site is fully browsable, searchable, and easy to customize. Scholars, librarians, and curators can build digital exhibits and collections without a database or backend.

- Builds static digital collection sites directly from IIIF Collections.
- Generates one page for each IIIF Manifest.
- Indexes Manifest metadata automatically for search and filtering.
- Provides clear, reusable IIIF-based components for presentation.
- Supports Markdown authoring for contextual and scholarly writing.
- Runs without a database or backend.
- Produces fast, lightweight, low-maintenance sites.
